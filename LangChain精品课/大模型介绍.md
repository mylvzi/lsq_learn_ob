## 1.CO-STAR结构化框架
>适用于`目标设定和问题解决`的场景下, 关键在于`清晰性和结构性`

![[Pasted image 20251228103437.png]]

* `Context` : 任务背景与上下文, 去关注`我为什么现在要问这个`, 所以这里要假设模型的角色
* `Objective` : 核心目标, 去关注`我希望这个回答帮助我解决什么决策或者困扰`
* `Steps` : 执行步骤, 去关注`你希望模型按照什么路径去思考问题`
* `TOne` : 语言风格, 去关注`我希望得到什么样的认知冲击`
* `Audience` : 目标用户, 去关注`我希望你把我当成什么样的人`
* `Respone` : 输出格式, 去关注`我想怎样使用你的回答`

# 三.嵌入模型
![[Pasted image 20251229225543.png]]

* ⽤数学的⽅式来⽐较向量，从⽽达到【度量语义】的⽬ 的
`度量语义的两种方式`
1. 欧氏距离 : 在同纬空间内部, 两种物体之间的实际距离表示两种物体的关系远近
2. 余弦相似度 : 关注向量的方向, 方向代表的是含义, 长度代表的是文本的长度

`应用场景`
* 语义搜索(Semantic Search) : 通过`相似度`这个概念,实现一种`模糊匹配`的方式, 通过计算向量间的相似度找到相关的内容
* 检索增强生成(Retrieval-Augmented Generation, RAG)
	检索增强 : 利用embedding model基于内部的知识库建立向量化数据库
	生成 : 根据相似度返回的结果, 利用LLM进行总结!(不是简单的返回结果, 省去了用户再点击一次的操作)
* 推荐系统 : 根据用户个人信息(喜好),利用嵌入模型, 推荐和用户喜好相似的物品,吸引用户
* 异常检测

### 方式二 : 使用代码计算语义距离
```python
import requests  
import numpy as np  
  
def get_embedding(text):  
    response = requests.post(  
        "http://localhost:11434/api/embeddings",  
        json={  
            "model": "nomic-embed-text",  
            "prompt": text  
        }  
    )  
    return np.array(response.json()["embedding"])  
  
def cosine_similarity(a, b):  
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))  
  
A = "地震灾害的遥感监测与应急响应"  
B = "地震灾害的可视化研究"  
C = "如何做意大利面才好吃"  
  
vec_A = get_embedding(A)  
vec_B = get_embedding(B)  
vec_C = get_embedding(C)  
  
print("A vs B:", cosine_similarity(vec_A, vec_B))  
print("A vs C:", cosine_similarity(vec_A, vec_C))
```

# 主流LLM的六大问题
## 1.模型具有幻觉
根本原因是: **语言训练的目标本就是最大化似然而不是精准构建**
LLM训练的核心目标 :
> 在给定的上下文条件下, 最大化下一个token出现的概率

`知识层面` : 模型只学习**相关性**, 并不理解**事实来源**
`数据层面` : 模型本身的训练语料库就是存在**污染,噪声的**, 劣质的语料库中存在大量的不合理/错误的描述,模型在生成答案时, 也会参考这些语料
`推理机制层面` : LLM是采用自回归的`逐token`产生答案的, 后面的结果依赖于前期的生成, 如果前期有一个`模糊`的token, 后面产生结果就会像滚雪球一样产生偏离更大的结果,就会形成**逻辑上完整,事实上错误的长文本**
## 2.切换模型
>模型与模型之间差异极大,如果要切换模型,原有的提示词/产出结果会发生大变化

## 3.专业问题不靠谱
对于比较专业性的问题, 我们只能对LLM的回答进行一种参考, 因为没有具体的`事实来源`
## 4.无法结构化输出输入
有些时候,我们希望模型的输入与输出是一种`接口`, 即输出内容时, 能以一种`结构化`的形式输出
比如统一以json/xml/md的格式输出内容

## 6.模型训练的语料库是有截止日期的

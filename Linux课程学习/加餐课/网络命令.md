# netstat : 检查网络状态
```bash
netstat -uapn
		-u:查看udp协议相关的网络
		-a:all
		-p:process,展示与网络相关的进程
		-n:number(localhost->127.0.0.1)
		-t:查看tcp协议相关的网络
		
watch -n 1 netstat -uapn   watch : 周期性执行网络命令
```

# 1.命令模式
yy : 复制当前行  n + yy : 向下复制n行(从当前光标开始)
p : 向下粘贴一行  n + p : 向下粘贴n行
u : 撤销  ctrl + r : 对历史撤销 撤销撤销
dd : 剪切光标所在行/删除所在行   dd + p 删除并粘贴  n + dd : 剪切 n 行


**光标定位相关**
$ : 定位到当前光标所在行末尾
^ : 定位到当前光标所在行首
gg : 定位到整个文本的最开头
G : 定位到整个文本的最后一行
n + G : 定位到第n行  
h : 左
j : 下
k : 上
l : 右 
hjkl 也 支持数字
w : 按单词跳转  定位到下一个单词的单词首字母
b : 和w相反
~ : 大小写切换
x : 删除光标后续的一个字符  也支持 n + x : 删除光标后续n个字符
X : 删除光标前面的一个字符
r : 对光标指定位置进行字符替换
R : 替换模式  直接进行文本的替换
ctrl w w : 切换到另一个编辑窗口   
ZZ  : 快速保存并退出

>命令模式是核心模式   无脑esc

# 2.底行模式
q : 直接退出  q! : 强制退出
wq : 保存并退出
set nu : 设置行号
set nonu : 去掉行号
/ 关键字 : 搜索关键字
! + 命令 : 不退出vim 执行shell命令   !man fgets : 不退出vim  直接查看fgets命令
vim不会默认创建文件,只有 wq之后才会创建文件
vs filename : 分屏,多文件操作

## 配置sudo
>sudo 进行提权   一般新建的用户不能直接sudo  必须添加进入 sudoers 文件之中 
>/etc/sudoers  相当于一个白名单
>能否执行sudo -- 本质就是在sudoers中进行用户的配置
>sudo 以 root用户的身份进行创建文件

## vim的配置
 原理 : vim的运行并不是直接点击可执行程序就运行的,在运行之前会读取 若干配置文件  最典型的就是 `.vimrc`文件,我们可以在.vimrc中添加自定义vim权限

# vim使用技巧总结
`块剪切`
```css
V   （选中当前行）注意这里是大写的V
j j （向下两行）
d   （剪切三行）
移动到想要粘贴的位置
p   （粘贴）
```
* 这里的核心步骤是 大写V : 起到一个块选中的作用
* 块缩进 : 选中后 按 `>`

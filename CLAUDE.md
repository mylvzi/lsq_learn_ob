## Project Overview

This is an **Obsidian knowledge management vault** used for personal learning and note-taking. It is not a code project—there is no build system, package manager, or executable code.

## Vault Structure

The vault contains learning materials organized by topic:

- **测试相关/** - Testing fundamentals (QA understanding, unit testing, agile development)
- **UI自动化学习/** - UI automation testing (Playwright)
- **比特精品课-接口自动化测试/** - API automation testing course
- **Linux课程学习/** - Linux OS course (processes, threads, signals, IO, file systems)
- **git精品课学习/** - Git version control
- **mysql精品课学习/** - MySQL database
- **Redis精品课学习记录/** - Redis database
- **LangChain精品课/** - LangChain/LangGraph AI framework
- **prompt 四周课程学习/** - Prompt engineering course
- **AI-Learn/** - AI learning materials
- **reading-diary/** - Reading notes (has separate git repo)
- **毕设记录/** - Graduation project records
- **项目实战练习/** - Practical project exercises
- **img/** - Image attachments

## Content Conventions

- **Wiki-links**: Use `[[note name]]` to link between notes
- **Language**: Content is primarily in Chinese (Simplified)
- **Attachments**: Stored in `img/` folder
- **Themes**: Uses "Things" theme with vim mode enabled

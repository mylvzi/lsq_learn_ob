>下面讨论的hash数据结构, 是value的一种数据类型  value 中的 hash数据类型 我们称之为 field - value(为了和key - value区分)

**hlen**
`hlen key` : 返回存储在key中的f的数量

**hsetnx**
`hsetnx key f v` : 给key 增加一对 f-v  如果 f 已经存在,返回0  并不会覆盖原有的数据
hset是在初试进行设置 k-v(f-v)
![[Pasted image 20251008204454.png]]
**hincrby**
`hincrby key f increment` : 给存储在key的f中的数字增加increment  如果key不存在,直接创建带有f的key  如果f不存在,直接创建f,并将其v设置为0

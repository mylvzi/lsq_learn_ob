# 1.基本介绍
mysqld是一个**服务端程序**,也被称为**mysql服务器**;是一个多线程程序(处理多个客户端的请求)
职责:对**数据目录**进行访问管理。数据目录 = mysql的主要工作目录 + 其他文件的默认存储位置(日志文件,状态文件等)

![[Pasted image 20250703215352.png]]
1. 连接管理:通过标识判断和哪一个客户端建立的链接
2. 处理sql语句:最基本的功能,完成查询操作
3. 内存管理:从**性能上**考量,对于一些频发查询的数据存储在内存之中,加快访问速度
4. 磁盘管理:持久化数据
5. 日志管理:存储操作记录。比如**wal**机制确保数据库的**一致性**
6. 集群管理等等
